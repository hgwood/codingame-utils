(inner) entity(strValue, …keys)

Parses a line of space-separated values then associates each value with the given keys, in order, into an object, which is returned. If a word can be parsed into a number, it is.
Parameters:
Name Type Attributes Description
strValue string a line of space-separated values
keys string <repeatable>
 keys of the resulting object
Source:
  • parse.js, line 31
Example
entity('1 2 c', 'a', 'b', 'c') // {a: 1, b: 2, c: 'c'}
